## REPOSITORY FILE DESCRIPTION:
**BankAccount:** in the BankAccount.java or BankAccount.java class file, you are going to find everything you need to create the setup of a conventional Bank account.

## *Description of the BankAccount.java class:* 
the BankAccount class is conformed by the following Attributes:

**ownerName:** Represents the variable where the name of the account owner is found. type string.
**ownerDocNumber:** Represents the variable of the account owner's document number. type int
**balance:** Represents the variable where the balance in the account will be stored, type int.
**accountNumber:** Represents the variable where the account number will be stored, type int.
**Activated:** Represents the variable to store if the account is active or inactive, type boolean.

## Processes carried out in the BankAccount class:
1. The constructor to the BankAccount class is created so that when desired it can be instantiated.
2. Get and set methods are created to obtain and set data such as account number, account status (active or inactive), available balance in the account and name of the account owner.
3. Create methods to deposit and withdraw balance.
