# Como ejecutar el programa:
Realizar la descarga del proyecto, dentro de su IDE favorito (Recomendamos Visual Studio Code) ingresar al terminal y ejecutar los siguientes comandos:

1. Ingresar a esta carpeta:
```
cd workshop6_dynamicpage
```

2. para instalar la carpeta node modules que contiene todas las dependencias instaladas de su proyecto, ingresar el siguiente codigo:

```
npm install
```

3. Para poner a correr la pagina ingresar al menu lateral Izquierdo:

```
boton Run and Debug
```
4. 
```
Dar clic en Launch Program

```
5. En el navegador de su preferencia ingresar a la siguiente dirección:

```
http://localhost:8080

```
6. Para verificar el consumo de la api:

```
Clic derecho en inspeccionar e ingresamos a la pestaña consola

```

7. Empezamos a dar clic en cada uno de los botones de la pagina principal de la aplicacion para visualizar el consumo

8. Si desea visualizar simplemente los archivos para ver la estructura del documentos en json colocamos la siguiente info en el navegador:

**visualizar la creacion del juego**
```
http://localhost:8080/api/v1/createGame

```
**visualizar el juego**
```
http://localhost:8080/api/v1/game

```

**visualizar el ganador del juego**
```
http://localhost:8080/api/v1/winnerGame

```

**visualizar el comienzo del juego**
```
http://localhost:8080/api/v1/startGame

```
