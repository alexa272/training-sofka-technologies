# How to Run the Program:
Download the project. In your favorite IDE (we recommend Visual Studio Code), open the terminal and run the following commands:

1. Navigate to this folder:
```
cd workshop6_dynamicpage
```

2. To install the node modules folder, which contains all the installed dependencies for your project, enter the following code:
```
npm install
```
3. To run the page, go to the left sidebar menu:
```
button Run and Debug
```
4.
```
Click on Launch Program
```
5. In your preferred browser, go to the following address:
```
http://localhost:8080
```
6. To check API usage:
```
Right-click on Inspect and go to the Console tab
```
7. Click on each of the buttons on the application's main page to view the usage
8. If you simply want to view the files to see the JSON document structure, enter the following information in your browser:
   
**View game creation**
```
http://localhost:8080/api/v1/createGame
```
**View the game**
```
http://localhost:8080/api/v1/game
```
**View the winner of the Game**
```
http://localhost:8080/api/v1/winnerGame
```
**View the start of the game**
```
http://localhost:8080/api/v1/startGame
```
